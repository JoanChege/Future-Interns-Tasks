Mental Health Chatbot

This project developed a mental health chatbot designed to assist individuals with depressive symptoms.
The chatbot leverages NLP and LSTM techniques to provide personalized and empathetic support.
It was built using the Flask framework and evaluated using standard metrics like accuracy, precision, recall, and F1-score.

Tour Guide App

A MERN Stack-powered web application was developed to serve as a virtual tour guide for Nairobi's tourist attractions.
It leverages MongoDB for data storage, Express.js for backend logic, React for frontend development, and Node.js as the runtime environment.
Key features include a comprehensive database of attractions, an interactive map, personalized recommendations, user reviews, and limited offline functionality.
The application aims to enhance the visitor experience by providing easy access to information and inspiring exploration. Future enhancements may include AR integration, virtual tours, and an AI-powered chatbot.
